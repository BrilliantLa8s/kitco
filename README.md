# Kitco Gem

This gem retrieves data (via HTTParty) from Kitco Charts about Gold and other precious metals.

Use at your own discretion 

## Install

    gem install kitco


## Examples

    require 'kitco'

    puts Kitco.gold
    puts Kitco.silver

## Command Line Utility


    kitco gold

For more details on run

    kitco --help
